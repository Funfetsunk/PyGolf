GOLF TILESET GUIDE
==================
Two files, each 256x192 (or 256x160) px, 16x16 source tiles, 16 columns wide.

How to load in the editor
--------------------------
1. Editor -> File -> Import Tileset PNG  -> golf_ground.png   (opaque, ground layer)
2. Editor -> File -> Import Tileset PNG  -> golf_detail.png   (RGBA, detail layer)
   Mark golf_detail.png as TRANSPARENT when loading.

Terrain logic codes (used in the Logic layer)
----------------------------------------------
  X = Tee Box     F = Fairway     R = Rough       D = Deep Rough
  B = Bunker      W = Water       T = Trees/OOB   G = Green

TIPS
----
- Use ground layer tiles to set the terrain coverage for the whole hole.
- Use detail layer sprites to add trees, rocks, flowers, markers on top.
- Edge tiles (cols 4-11 in each row) help smooth the boundary between
  two different terrain types.
- Set the Logic layer to match the terrain: paint G under the green,
  F under fairway, etc. The game reads the logic layer for gameplay.
- The flag + hole cup sprites go on the detail layer at the pin position.
  The logic layer at that same cell should be G (green).
- Tee box sprites go on the detail layer at the tee position.
  The logic layer at that cell should be X (tee box).
